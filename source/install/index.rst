Въведение
=========

Go е проект с отворен код с BSD-лиценз. Съществуват два официални Go набора от компилатори: ``gc`` - Go компилатор и
``gccgo`` компилатор, който е част от GNU C Cmpiler (GCC).

``gc`` е по-напредналия и по-добре тестван от двата. Тази страница е относно инсталирането на изпълнимата дистрибуция
на ``gc`` компилатора.

За информация относно инсталиране на ``gc`` компилатора от изходен код, вижте :doc:`source_install`. За повече
информация относно инсталирането на ``gccgo``, вижте :doc:`gccgo_install`.

Сдобиване с go инструментите
============================

Посетете `Go projects's download page <http://code.google.com/p/go/downloads>`_ и изберете изпълнимата дистрибуция, 
която отговаря на вашата операционна система и архитектура на процесора.

Официални изпълними дистрибуции са налични за FreeBSD, Linux, Mac OS X и Windows операционни системи и 32-bit 
(``386``) и 64-bit (``amd64``) x86 процесорни архитектури.

Ако няма изпълнима дистрибуция за вашата операционна система/архитектура комбинация можете да опитате 
:doc:`source_install` или :doc:`gccgo_install`.

Инсталиране на go инструментите
===============================

Go изпълнимите дистрибуции приемат, че ще бъдат инсталирани в :file:`/usr/local/go`, но е възможно да ги инсталиране и 
на различно място. Ако изберете последното трябва да зададете ``GOROOT`` променлива в средата, която да сочи към
директрията с Go инструментите.

На пример, ако сте иснталирали Go във вашата домашна директория, трябва да добавите следните редове в 
``$HOME/.profile``:

.. code-block:: console

    export GOROOT=$HOME/go
    export PATH=$PATH:$GOROOT/bin

FeeBSD и Linux
--------------

Разархивирайте архива в :file:`/usr/local`, създавайки Go сърво в file:`/usr/local/go` (обикновено това трябва да бъде
изпълнено като **root** или чрез ``sudo``):

.. code-block:: console
    
    tar -C /usr/local go.release.go1.tar.gz

Добавете :file:`/usr/local/go/bin` във PATH променливата на средата. Можете да направите това, добавяйки следния ред
във вашия :file:`/etc/.profile` (при инсталация за всички потребители) или :file:`$HOME/.profile`:

.. code-block:: console

    export PATH=$PATH:/usr/local/go/bin

Mac OS X
--------

Отворете ``.pkg`` файла и следвайте инструкциите за инсталиране на Go инструментите. Пакетът се инсталира в 
:file:`/usr/local/go`.

Пакетът би трябвало да сложи :file:`/usr/local/go/bin` директорията във вашата ``PATH`` променлива на средата. Може да
се наложи да рестартирате оворените сесии на Terminal, за да бъдат приложени промените.

Windows
-------

.. warning:: Все още няма инструкции за инсталация под Windows

Тестване на инсталацията
========================

Проверете дали Go е правилно инталиран, като построите простра програма, както следва.

Създайте файл :file:`hello.go` и поставете в него следната програмата:

.. code-block:: go

    package main

    import "fmt"

    func main() {
        fmt.Printf("Здравей, свят!\n")
    }

След това изпълнете с ``go`` инструмента:

.. code-block:: console

    $ go run hello.go
    Здравей, свят!

Ако видите съобщението "Здравей, свят!", означава, че Go инсталацията работи.
